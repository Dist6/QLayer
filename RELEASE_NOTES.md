# QLayer v0.1.0

First public Windows release.

## Highlights

- Push-to-talk Voice Flow for Codex and ChatGPT
- Background audio Off, Lower, Mute, and Restore
- Chat Shortcuts and recent-chat discovery
- Localhost Manager
- Projects with linked chats and preferred ports
- Windows tray utility
- Local-first preferences with no account or API key

## Platform

- Windows 10/11
- x64
- Portable ZIP; no installer or administrator access required

## Known limitations

- Unsigned early build; Windows SmartScreen may display a warning
- Windows only
- Microsoft Edge WebView2 Runtime is required
- No automatic updates
- No direct development process control
- Moving the executable can invalidate an existing launch-at-startup entry
