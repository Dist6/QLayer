# Privacy

QoLayer is local-first.

v0.1 does not:

- Collect telemetry.
- Record audio.
- Upload prompts.
- Collect credentials or tokens.
- Read Codex auth files.
- Read browser cookies.
- Proxy traffic.
- Add remote logging.
- Add cloud sync.
- Expose arbitrary keyboard input or text typing.

v0.1 stores only local preferences. Future add-ons that require extra permissions must be explicit and opt-in.
